  팀원이 처음 clone 받는 법 (필요 시 안내)
  ═══════════════════════════════════════════

  1) git clone https://github.com/hackathoncnu/frontend.git
  2) cd frontend
  3) npm install
  4) .env.local 새로 만들기:
     VITE_KAKAO_REST_API_KEY=각자_받은_REST_키
  5) npm run dev
